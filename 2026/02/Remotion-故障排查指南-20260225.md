# Remotion 故障排查指南

常见问题诊断和解决方案

---

## 渲染问题

### 黑屏问题

**症状**：输出视频是全黑

**原因**：
1. 组件未正确渲染
2. 背景和文本颜色相同
3. 时间范围超出范围

**解决**：
1. 在浏览器中预览检查
2. 检查 CSS 颜色值
3. 检查 `durationInFrames` 设置

### 内存溢出

**症状**：渲染崩溃，"FATAL ERROR: Allocation failed"

**原因**：
1. 处理大视频文件
2. 过多并发进程
3. 内存泄漏

**解决**：
1. 使用 `OffthreadVideo`
2. 减少并发数（从 4 改为 2）
3. 增加系统内存或使用分段渲染

---

## 性能问题

### 渲染缓慢

**症状**：渲染速度 < 1x 实时速度

**优化**：
1. 降低分辨率进行测试
2. 检查是否有复杂动画
3. 使用 Performance 工具分析

### 字体加载缓慢

**症状**：首帧延迟长

**解决**：
1. 预加载字体
2. 使用系统字体而非 Google Fonts
3. 减少字体文件数量

---

## 音频问题

### 音频和视频不同步

**症状**：字幕和配音错开

**解决**：
```typescript
// 使用精确的时长计算
const duration = await getAudioDurationInSeconds(audioFile);
const durationInFrames = Math.round(duration * fps);
```

### 音量不一致

**解决**：使用 ffmpeg 标准化音量

---

## 导出问题

### 文件过大

**解决**：
1. 增加 CRF 值（质量降低但文件更小）
2. 使用 H.265 编解码器
3. 降低分辨率或帧率

### 兼容性问题

**解决**：
- 使用 H.264 编解码器（兼容性最好）
- 测试在多个平台上
