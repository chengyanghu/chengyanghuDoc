## 性能最佳实践

### 1. 内存优化
- 使用 `OffthreadVideo` 处理大视频
- 及时释放不使用的资源
- 使用虚拟化处理长列表

### 2. 渲染优化
- 使用 `memo` 包装不常变化的组件
- 避免在渲染中创建新对象
- 使用 CSS 变量替代动态计算

### 3. 开发效率
- 使用快速预览模式（低分辨率）
- 分段渲染而不是全量渲染
- 缓存中间产物
