# OpenCode 模型搭配推荐

> **研究概况**
> - 研究来源：Context7 官方文档
> - 报告生成：2026-02-26

## 核心发现

OpenCode 官方明确指出：模型很多，但**同时擅长代码生成和工具调用**的模型很少 [[1]](https://github.com/anomalyco/opencode/blob/dev/packages/web/src/content/docs/models.mdx)。

---

## 推荐模型列表

### 🥇 最高推荐（OpenCode 官方验证）

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| **GPT 5.2** | 最新旗舰，代码能力极强 | 复杂项目开发 |
| **GPT 5.1 Codex** | 专门优化代码生成 | 高强度编码任务 |
| **Claude Opus 4.5** | 最强推理能力 | 复杂架构设计 |
| **Claude Sonnet 4.5** | 平衡性价比 | 日常开发任务 |
| **Minimax M2.1** | OpenCode 深度优化 | 推荐首选 |
| **Gemini 3 Pro** | 多模态能力 | 复杂项目 |

> OpenCode 官方推荐："GPT 5.2, GPT 5.1 Codex, Claude Opus 4.5, Claude Sonnet 4.5, Minimax M2.1, and Gemini 3 Pro" [[1]](https://github.com/anomalyco/opencode/blob/dev/packages/web/src/content/docs/models.mdx)

---

## OpenCode Zen 精选模型

OpenCode 团队测试验证过的模型列表 [[2]](https://github.com/anomalyco/opencode/blob/dev/packages/web/src/content/docs/zen.mdx)：

### GPT 系列（via OpenCode Zen）
- `gpt-5.2-codex` - 最佳代码模型
- `gpt-5.1-codex-max` - 高性能代码
- `gpt-5.1-codex-mini` - 轻量快速

### Claude 系列
- `claude-opus-4-20250514` - 顶级推理
- `claude-sonnet-4-20250514` - 平衡之选

### MiniMax 系列（推荐）
- `minimax-m2.5` - OpenCode 深度优化
- `minimax-m2.5-free` - 免费版
- `minimax-m2.1` - 高性价比

### 国内模型
- `glm-5` - 智谱最新
- `kimi-k2.5` - 月之暗面
- `qwen3-coder` - 阿里代码专用

---

## 配置示例

```json
{
  "model": "anthropic/claude-sonnet-4-20250514"
}
```

或使用 OpenCode Zen：

```json
{
  "model": "opencode/minimax-m2.5"
}
```

---

## 选择建议

| 场景 | 推荐模型 |
|------|----------|
| 追求最高性能 | GPT 5.2 Codex / Claude Opus 4.5 |
| 日常开发 | Claude Sonnet 4.5 / Minimax M2.1 |
| 预算有限 | Minimax M2.5 Free / GPT 5 Nano |
| 国内用户 | Minimax M2.1 / GLM-5 / Kimi K2.5 |
