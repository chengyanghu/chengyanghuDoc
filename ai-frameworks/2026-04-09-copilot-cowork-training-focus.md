# Copilot Cowork 培训研究素材 — 2026-04-09

> **用途**：供企业内训负责人编写 Copilot Cowork 培训材料时的参考底稿
> **目标受众**：公司全员（非技术背景为主）
> **核心目标**：让员工理解"这是什么→跟我有什么关系→怎么用→注意什么"

---

## 一、30 秒讲清楚 Copilot Cowork

### 给员工的一句话解释

> **以前的 Copilot**：你问一个问题，它给一个回答。你需要自己把回答复制到邮件、PPT、Excel 里。
>
> **现在的 Copilot Cowork**：你说一个目标（比如"帮我准备下周三的客户会议"），它**自己去做**——查邮件、看 Teams 记录、找文件、做 PPT、排日程——你只需要在关键节点点"同意"或"改一下"。

### 类比

| 旧版 Copilot | Copilot Cowork |
|:---:|:---:|
| 像一个聪明的实习生 | 像一个能干的助理 |
| 你问什么答什么 | 你说目标，它安排全流程 |
| 一问一答，一次一件事 | 多步骤、跨应用、后台持续运行 |
| 产出：一段文字/一个草稿 | 产出：邮件已发、会议已排、PPT 已建 |

## 二、员工最关心的 5 个问题

### Q1：什么时候能用？

| 阶段 | 时间 | 状态 |
|------|------|------|
| 私有预览 | 2026-03-09 | 仅限特定企业 |
| Frontier 预览 | 2026-03-30 | 企业可申请加入 |
| E7 正式上线 | 2026-05-01（计划）| 付费订阅 |
| 全面可用 (GA) | 待定 | 未公布 |

**培训要点**：目前处于 Frontier 预览阶段，需要公司 IT 管理员启用。员工无法自行开通，需等公司统一部署。

### Q2：我需要学什么新技能？

**关键认知转变**：从"向 AI 提问"到"给 AI 委派任务"。

| 旧习惯（要改） | 新习惯（要学） |
|----------------|----------------|
| "帮我写一封邮件" | "帮我准备下周的客户会议，包括邮件、PPT 和日程" |
| 一次只做一件事 | 描述最终目标，让 AI 规划步骤 |
| 自己在各个 App 之间切换 | 让 Cowork 跨 App 协作 |
| 拿到草稿后手动修改 | 在检查点审批或调整方向 |

**培训核心能力**：
1. **目标描述能力** — 清晰表达"我要什么结果"而非"我要你做什么动作"
2. **检查点审批意识** — 知道在哪些节点需要检查 AI 的工作
3. **数据卫生意识** — 理解 AI 只能基于它能访问的数据工作

### Q3：它能做什么？

#### 培训素材建议：按岗位分场景

**行政/助理岗**
- 日历优化：自动分析一周日程，识别低价值会议，建议改期/取消，预留专注时间
- 会议准备：自动搜集相关邮件、文件、历史会议记录，生成简报
- 邮件处理：批量起草回复、转发带附件的邮件

**市场/销售岗**
- 客户会议全套准备：邮件历史 → 竞品分析 Excel → 演示 PPT → 日程安排
- 产品发布：竞争分析表、定位文档、演示材料、里程碑规划
- 公司研究：自动拉取财报、监管文件、分析师评论，生成综合报告

**项目管理岗**
- 项目状态汇总：跨 Teams 频道和邮件收集进展，生成周报
- 利益相关方沟通：自动起草不同层级的沟通邮件
- 数据分析：从 Excel/SharePoint 数据生成报告和可视化

**通用场景（所有人）**
- 每日简报：自动生成今日重点邮件、会议、待办摘要
- 深度研究：跨多个来源收集信息，编写分析报告
- 文档创建：Word、Excel、PowerPoint、PDF 一步到位

### Q4：安全吗？会不会发错邮件？

**培训重点（打消顾虑）**：

1. **所有敏感操作需你亲自批准** — 发邮件、改日历、发 Teams 消息前，Cowork 会暂停等你确认
2. **全程可见** — 每个步骤都在对话窗口中展示，不是"黑箱"
3. **随时可停** — 你可以在任何时候暂停、修改方向或取消
4. **不确定就问** — 如果 Cowork 不确定你的意图，会主动向你提问
5. **企业数据保护** — 所有操作在公司的 Microsoft 365 安全边界内

**审批流程图**：

```
你的请求 → Cowork 制定计划 → [你审批计划]
                                    ↓
              步骤1: 搜集资料 → 步骤2: 生成草稿
                                    ↓
              [你审批草稿] → 步骤3: 发送邮件 → [你审批发送]
                                    ↓
                               ✅ 完成
```

> **每个 [方括号] = 你的审批点，AI 不会跳过**

### Q5：跟现在的 Copilot 有什么区别？

| 维度 | 现有 Copilot | Copilot Cowork |
|------|-------------|----------------|
| 交互模式 | 一问一答 | 目标驱动，多步执行 |
| 运行时间 | 秒级响应 | 分钟到小时级 |
| 跨应用 | 单个 App 内 | 跨 Outlook/Teams/Excel/Word/PPT |
| 产出物 | 文本/草稿 | 已发邮件、已建文档、已排会议 |
| 审批 | 无（直接给结果） | 关键节点需人工审批 |
| 底层模型 | OpenAI GPT | Anthropic Claude（+ GPT 按需路由）|

## 三、培训设计建议

### 3.1 建议培训结构

```
模块 1（15 分钟）：认知刷新
  ├── 什么是 Copilot Cowork（演示视频：1分49秒官方视频）
  ├── 与旧版 Copilot 的区别（对比表）
  └── 核心认知转变："提问"→"委派"

模块 2（20 分钟）：场景演练
  ├── 场景 A：日历优化（全员通用）
  ├── 场景 B：会议准备（按岗位分组）
  └── 场景 C：数据分析和报告（进阶）

模块 3（10 分钟）：安全与规范
  ├── 审批机制讲解
  ├── 公司数据使用政策
  └── DO's 和 DON'Ts

模块 4（15 分钟）：实操练习
  ├── 每人尝试一个真实场景
  └── Q&A
```

### 3.2 培训前准备清单（给 IT/培训团队）

- [ ] 确认公司是否已加入 Frontier 预览计划
- [ ] IT 管理员在 M365 Admin Center → Agent 管理中启用 Cowork
- [ ] 确定首批试用人群范围（建议从行政、市场、项目管理岗开始）
- [ ] 审查公司数据权限设置（Cowork 能访问员工有权限的所有数据）
- [ ] 准备培训环境：每人有 M365 Copilot 许可证
- [ ] 制定公司级 AI 使用规范（哪些操作允许自动执行、哪些必须人工审批）

### 3.3 DO's 和 DON'Ts 卡片

#### DO's ✓

| 做法 | 原因 |
|------|------|
| 描述**最终目标**而非步骤 | Cowork 擅长自己规划路径 |
| 每次审批前**仔细检查**内容 | AI 可能误解你的意图 |
| 从**低风险场景**开始练习 | 如日历整理、资料汇总 |
| 提供**充足上下文** | "准备下周三与 ABC 公司的季度评审会议"优于"准备会议" |
| 反馈不满意时**调整方向** | 不必从头开始，可在检查点修正 |

#### DON'Ts ✗

| 做法 | 风险 |
|------|------|
| 不看内容就点"批准" | 可能发出不当邮件或安排错误会议 |
| 用于处理**机密/敏感数据** | 预览阶段功能可能变化，谨慎为上 |
| 期望 AI 理解**公司政治** | 它不知道某个会议不能取消或某人不能 CC |
| 一次委派**过于复杂**的任务 | 先从 2-3 步的任务开始，逐步信任 |
| 忽视**数据整理** | 你的文件命名混乱 = AI 找不到正确文件 |

## 五、培训常见问答预备

**Q: Cowork 会替代我的工作吗？**
> 不会。Cowork 处理的是重复性的协调工作（排会议、整理资料、写初稿），让你有更多时间做需要判断力和创造力的事。它是"数字助理"，不是"替代者"。

**Q: 它做错了怎么办？**
> 每个关键操作都需要你批准。如果你在审批时发现问题，直接修改指示或取消即可。已发出的邮件等同于你自己发出的邮件——所以审批时要认真检查。

**Q: 它能看到我所有的邮件和文件吗？**
> Cowork 的访问权限和你现有的 M365 权限一致。它能看到你能看到的内容，不能看到你无权访问的内容。公司的数据保护策略同样适用。

**Q: 我需要用英文跟它交流吗？**
> 目前 Frontier 预览以英文为主，中文支持程度有限。正式版会逐步扩展语言支持。建议培训时先以英文 prompt 为主。

**Q: 跟 ChatGPT/Claude 网页版有什么区别？**
> 最大区别：Cowork 直接嵌入你的工作环境（Outlook、Teams、Excel），能直接操作你的邮件、日历、文件。ChatGPT/Claude 网页版只能在对话框里给你文字回答，你还得自己去各个 App 操作。

**Q: 我已经在用 Copilot 了，还需要学新东西吗？**
> 核心操作界面没变（还是在 Copilot Chat 里），但**使用方式变了**——从"提一个问题"变成"委派一项工作"。最大的学习曲线是：学会描述"我要什么结果"而不是"你帮我做这一步"。
