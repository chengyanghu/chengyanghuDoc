# 上海二手房装修 · 全流程对话 Checklist

> 适用场景：与装修公司谈判、签约、施工全程
> 每项打 ✅ 确认，对方回答模糊或拒绝谈判 → 重新评估合作意愿

---

## 第一关：签约前——公司资质核查

- [ ] **营业执照**：在[国家企业信用信息公示系统](https://www.gsxt.gov.cn)搜索，确认有效期和经营范围含"建筑装饰装修"
- [ ] **装饰资质证书**：在[全国建筑市场监管平台](https://jzsc.mohurd.gov.cn)查"建筑装饰装修工程专业承包"资质等级（壹级最高）
- [ ] **司法记录**：在[中国裁判文书网](https://wenshu.court.gov.cn)搜公司名，近3年无重大合同纠纷判决
- [ ] **失信记录**：在[中国执行信息公开网](https://zxgk.court.gov.cn)查，无"失信被执行人"
- [ ] **成立年限 ≥ 3 年**，有实体门店，有正式收款账户（非个人微信）
- [ ] **实地考察**近期竣工项目（同小区/同面积），非样板间

## 第二关：谈判阶段——逐项确认（现场问+记录回答）

### 【公司与团队】

| 问题 | 对方回答 |
|------|---------|
| 本项目项目经理是谁？施工全程是否固定不换人？ | |
| 是否分包/转包给第三方？分包方有什么资质？ | |
| 主要工种（水电、泥工、木工、油漆）是否自有团队？ | |

### 【报价单】

| 问题 | 对方回答 |
|------|---------|
| 报价中每项工程量怎么计算的？墙面面积是否扣除门窗？ | |
| 辅材的品牌和规格是什么？能否写入合同附件？ | |
| 报价单中有哪些"暂定价"项目？实际如何结算？ | |
| 预计增项来自哪些方面？历史项目平均增项比例是多少？ | |
| 增项上限能否约定为合同总价的 10%，超出须甲方书面授权？ | |
| 水电改造是否可以按实际竣工量结算（附竣工图）？ | |

### 【付款节点】

| 问题 | 对方回答 |
|------|---------|
| 首付款比例能否降到合同总价 10%，进场后再付 20%？ | |
| 每笔付款是否与阶段验收挂钩（验收通过后才付，非按时间）？ | |
| 能否保留 10% 质保金，竣工满 12 个月后支付？ | |
| 是否接受第三方资金监管账户付款（大额项目）？ | |

### 【工期与违约】

| 问题 | 对方回答 |
|------|---------|
| 开工日期和竣工日期能否写明具体年月日（非"约X工作日"）？ | |
| 乙方逾期完工，违约金能否约定为合同总价 3‰/天？ | |
| "不可抗力"具体包含哪些？材料涨价/工人不足是否算在内？ | |
| 逾期超多少天，甲方有权单方面解除合同？ | |

### 【材料管理】

| 问题 | 对方回答 |
|------|---------|
| 所有主材品牌型号是否可写入合同附件《材料确认单》？ | |
| 材料进场时是否需要甲方到场核验，未确认不得使用？ | |
| 发现材料与合同不符，赔偿方案是什么（建议差价3倍）？ | |
| 业主委托的第三方监理是否可随时进场抽检材料？ | |

### 【保修条款】

| 问题 | 对方回答 |
|------|---------|
| 防水工程保修几年（法定5年，目标8年）？ | |
| 水电管线保修几年（法定2年，目标5年）？ | |
| 保修期内出现问题，响应时限和维修时限各几天？ | |
| 公司若在保修期内停业，保修责任由谁承担？ | |

### 【合同条款】

| 问题 | 对方回答 |
|------|---------|
| 合同是否可以签《补充协议》修改格式条款？ | |
| 是否可以约定增项上限（超合同总价15%须特别授权）？ | |
| 纠纷解决能否约定为向工程所在地法院诉讼（非仲裁）？ | |
| 甲方有因解除合同（逾期/转包/材料不符）无需支付违约金，是否接受？ | |
| 口头承诺须以书面补充协议确认，否则不具法律效力，是否接受？ | |

## 第三关：合同审查——签字前逐项勾选

- [ ] 乙方盖公司公章（非个人签字）
- [ ] 付款条件写的是"验收通过后"（非"完工后X天内"）
- [ ] 开工日期、竣工日期均为具体年月日
- [ ] 违约金：乙方逾期 ≥ 3‰/天；甲方逾期 ≤ 1‰/天（双向对等）
- [ ] 《工程量清单》已附入合同，每项含：单位、数量、单价、工艺说明
- [ ] 《材料确认单》已附入合同，主材品牌/型号/规格均已填写
- [ ] 各工种保修年限已明确列出（防水单独列）
- [ ] 增项上限条款已写入
- [ ] 10% 质保金保留至竣工后12个月
- [ ] 隐蔽工程验收须甲方到场条款已写入
- [ ] 禁止转包条款已写入（或分包须甲方书面同意）
- [ ] 纠纷解决方式为法院诉讼（非仲裁）

## 第四关：施工过程——各节点验收 Checklist

### 材料进场验收
- [ ] 对照《材料确认单》核对品牌/型号/规格
- [ ] 扫描品牌防伪码验真（多乐士、立邦均有官方验真渠道）
- [ ] 拍摄包装标签+批号+进场全景照，微信存档（带时间戳）

### 水电隐蔽验收（封槽前必须完成）
- [ ] 冷热水管打压测试：0.8MPa 保压 30 分钟，压力不降
- [ ] 强弱电分管分槽，无交叉
- [ ] 拍摄管材品牌字样、壁厚、接头处理
- [ ] 甲方在《水电隐蔽验收单》上签字后，方可封槽

### 瓦工验收
- [ ] 空鼓率 ≤ 5%（用小锤敲击，闷响=空鼓）
- [ ] 卫生间闭水试验 48 小时无渗漏，楼下业主确认签字
- [ ] 瓷砖平整度偏差 ≤ 2mm（2m 靠尺检测）

### 木工验收
- [ ] 柜门开合顺畅，铰链不跑位
- [ ] 对角线误差 ≤ 2mm
- [ ] 背板材料与合同一致（确认厚度）

### 涂料验收
- [ ] 墙面平整度偏差 ≤ 3mm（2m 靠尺斜照检测）
- [ ] 无色差、无流坠、无批刮痕迹
- [ ] 底漆遍数与合同一致

### 竣工验收
- [ ] 乙方提交水电竣工图（含管线走向、插座位置、回路说明）
- [ ] 乙方提交主要材料合格证/检测报告
- [ ] 全屋无遗留问题，问题清单整改完毕
- [ ] 空气质量检测（建议竣工通风3个月后，委托CMA资质机构检测甲醛/苯/TVOC）
- [ ] 甲方在《竣工验收单》上签字（签字代表验收通过，质保期起算）

## 付款节点速查表

| # | 付款节点 | 比例 | 前提条件 |
|---|---------|------|---------|
| 1 | 签约首付 | 10% | 合同双方签字盖章 |
| 2 | 进场款 | 20% | 材料进场验收通过 |
| 3 | 水电完工款 | 15% | 水电隐蔽验收通过 |
| 4 | 瓦工完工款 | 15% | 闭水试验48h通过 |
| 5 | 木工完工款 | 15% | 木作验收通过 |
| 6 | 油漆完工款 | 10% | 涂料验收通过 |
| 7 | 竣工款 | 10% | 竣工验收单签字 |
| 8 | **质保尾款** | **5%** | **入住满12个月** |

**黄金原则：尾款合计 ≥ 10%；每笔付款必须绑定验收，不按时间节点付款。**

## 红旗信号——遇到立刻重新评估

| 信号 | 风险 |
|------|------|
| 首付要求 ≥ 40% | 卷款跑路高风险 |
| 拒绝任何合同修改 | 格式条款对业主极度不利 |
| 样板间无法实地参观 | 没有真实完工项目 |
| 报价低于市场价 20% 以上 | 后期增项/材料偷换补利润 |
| 要求微信个人账户转账 | 无正规收款，维权困难 |
| 拒绝第三方监理进场 | 违约即为默认 |
| "不可抗力"包含材料涨价 | 合同对乙方极度有利 |
